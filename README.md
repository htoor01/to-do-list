# To-Do List Web App

A simple, interactive **to-do list** built with **HTML, CSS, and JavaScript**, featuring task management, persistence via localStorage, and user-friendly controls. Note: stored locally.

## Features

- **Add Tasks:** Enter a task in the input field and press **Add** or **Enter**  
- **Mark Completed:** Click on a task to toggle **strikethrough**  
- **Delete Single Task:** Click the **×** on the right side of a task  
- **Clear All Tasks:** Clear the entire list with one click  
- **Persistent Storage:** Tasks are stored locally and displayed on page reload  

## Technologies Used

- **HTML** – Structure and semantic layout  
- **CSS** – Styling, flex/grid layout, responsive design  
- **JavaScript** – DOM manipulation, event handling, localStorage  

## How to Use

1. Clone or download the repository.  
2. Open `index.html` in your browser.  
3. Add, complete, delete, or clear tasks as desired.  
4. Tasks are saved automatically in your browser’s local storage.
